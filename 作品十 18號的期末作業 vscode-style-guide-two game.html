## 快速開始

git clone https://github.com/wen-pin/project.git cd project cp .env.example .env docker-compose up -d php artisan migrate

## 技術棧

  • PHP 8.3 + Laravel 11
  • Node.js 20 + React 18
  • MySQL 8 + Redis
  • Docker + Docker Compose
